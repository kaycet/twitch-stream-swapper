TSR Icon Generator

Generates icon-16.png, icon-48.png, icon-128.png, and icon-512.png from icon-source.svg.

If downloads fail due to browser restrictions on file://, run a local server in the repo root: python -m http.server 8000 then open http://localhost:8000/icons/generate-icons.html.

Output filenames match what the extension expects in manifest.json.

Tips for crisp icons:

- Keep shapes bold, avoid tiny details
- Use a safe padding so nothing touches the edge
- Test at 16px and 48px — if it reads there, it’ll read anywhere
